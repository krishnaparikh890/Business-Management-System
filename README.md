
# ERP SYSTEM 

## Overview
This is a comprehensive Enterprise Resource Planning (ERP) system designed to streamline business operations. The application provides a centralized platform to manage core business processes, helping organizations improve efficiency, data accuracy, and decision-making.

## Features
- Dashboard Management: A high-level overview of key business metrics and reports.
- User Authentication: Secure login system with role-based access control.
- Responsive UI: A clean, professional UI designed for consistency and ease of use.

## Tech Stack

### **Frontend**

- **React.js -** For building the user interface.
- **Tailwind CSS -** For responsive and customizable styling.

## Steps to run the Application

1. Clone the repository:
```
git clone https://github.com/krishnaparikh890/ERP-System.git

```
### Frontend Setup

1. Navigate to the frontend directory:
```
cd ../frontend
```
2. Install dependencies:
```
npm install
```
3. Start the frontend server:
```
npm Run dev
```

## Usage
- Log in to the system using your credentials.
- Navigate through the sidebar to access different business modules.
- Use the management panels to view, add, or update business records.
